# mock-time-postOffice

---
## linux部署/ZeroTermux本地部署

1. 换源

```
自行更换清华源
```


1. 克隆代码到本地

```
pkg install git
git clone https://github.com/chengzhnag/mock-time-postOffice.git
```

2. 安装python

```
pkg install python
```

3. 安装模块

```
.env.development  // 开发
.env.production  // 生产
修改文件中的'VUE_APP_BASE_API'配置
ps: 后端接口默认端口3000
```

4. 运行

```
// mock-time-postOffice目录下运行
npm run
```
